JavaScript Variable Declaration Exercises

Author: Nichelle Barton

Date: October 13, 2025

Course: Web124-2025

Exercise Instructions

In your JavaScript file, declare the following items to store data. After each declaration, add a console.log statement to review the data in the browser console.

Questions and Answers

Question 1:

A variable to store your full name

Answer:

// Declare variable to store full name
let fullName = "Nichelle Barton";
console.log("Full Name:", fullName);

Question 2:

A variable to store your desired annual salary

Answer:

// Declare variable to store desired annual salary
let desiredAnnualSalary = 85000;
console.log("Desired Annual Salary:", desiredAnnualSalary);

Question 3:

A variable to store your veteran status (hint: either you are a veteran or you are not a veteran)

Answer:

// Declare variable to store veteran status (boolean value)
let isVeteran = false;
console.log("Veteran Status:", isVeteran);

Question 4:

An array to store the names of three of your friends

Answer:

// Declare array to store names of three friends
let friendsNames = ["Sarah Johnson", "Michael Chen", "Emma Rodriguez"];
console.log("Friends Names:", friendsNames);

Question 5:

An array to store the value of the desired annual salary for your three friends

Answer:

// Declare array to store desired annual salaries for three friends
let friendsSalaries = [75000, 92000, 68000];
console.log("Friends Desired Salaries:", friendsSalaries);

Question 6:

A literal object to store the first name, last name, and desired annual salary of yet another friend

Answer:

// Declare literal object to store another friend's information
let anotherFriend = {
    firstName: "David",
    lastName: "Thompson",
    desiredAnnualSalary: 78000
};
console.log("Another Friend:", anotherFriend);

Implementation

Note: The JavaScript implementation can be found in the linked script.js file. Open your browser's developer tools and check the console to see the output of all console.log statements.

How to View Console Output:

  1. Open this page in your web browser
  2. Right-click on the page and select "Inspect" or "Developer Tools"
  3. Click on the "Console" tab
  4. You should see all the variable declarations and their values logged to the console

Learning Objectives

This exercise demonstrates:

  • Variable declaration using let
  • Different data types: string, number, boolean
  • Array creation and initialization
  • Object literal syntax and properties
  • Console logging for debugging and data review
